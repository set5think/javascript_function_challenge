Graphviz plugin

Inline graphviz

You can embed graphviz code to generate graphs

:inlinegraphviz [engine=neato]
graph test123 {
	a -- b -- c;
	a -- {x y};
	x -- c [w=10.0];
	x -- y [w=5.0,len=3];
}
:endinlinegraphviz

:inlinegraphviz
graph test123 {
a — b — c;
a — {x y};
x — c [w=10.0];
x — y [w=5.0,len=3];
}
:end

File based graphviz

It is also possible to process graphviz from a given file:

:graphviz code/graphviz/graph.dot

:graphviz code/graphviz/graph.dot
